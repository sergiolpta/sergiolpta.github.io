Ferramenta de Minificação

Esta ferramenta gera versões minificadas dos arquivos CSS e JavaScript do site, reduzindo seu tamanho e melhorando o desempenho de carregamento.

Arquivos Minificados

Clique nos links abaixo para baixar as versões minificadas dos arquivos:

  • Carregando arquivos...

Como usar os arquivos minificados:

  1. Baixe os arquivos minificados clicando nos links acima
  2. Faça upload dos arquivos para o diretório correspondente no seu servidor
  3. Atualize as referências nos arquivos HTML para apontar para as versões minificadas
  4. Por exemplo, mude <link rel="stylesheet" href="styles4.8.css"> para <link rel="stylesheet" href="styles4.8.min.css">
